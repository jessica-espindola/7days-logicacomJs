Descrição do Desafio

O objetivo principal deste desafio é atualizar o programa do 5º Desafio, inserindo a possibilidade do usuário remover um item da lista.

Para tal, deverá aparecer pro usuário uma opção para ele remover algum item da lista, exibido na pergunta você deseja adicionar uma comida na lista de compras?

Caso o usuário escolha a opção de remover o item, o programa deverá imprimir os elementos presentes na lista atual e a pessoa deverá escrever qual deles deseja remover.

Depois disso, o programa irá remover o elemento da lista e imprimir a confirmação de que o item realmente não está mais lá. Por fim, o programa voltará para o ciclo inicial de perguntas.

Se na hora de deletar o item, o mesmo não for encontrado na lista, o programa deverá exibir uma mensagem avisando: "Não foi possível encontrar o item dentro da lista!"

OBS: A opção de remover um item só deverá estar disponível a partir do momento que existir ao menos um elemento dentro da lista de compras.
