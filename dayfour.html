Descrição do Desafio

O objetivo principal deste desafio é criar um "Jogo de Adivinhação", que comece com um valor específico entre 0 a 10 e o usuário tentará adivinhar qual é esse número...

Regras do jogo:

  • O programa precisa perguntar ao usuário qual número ele gostaria de chutar para adivinhar o número escolhido;
  • Caso o usuário acerte, o programa deverá parabenizar o usuário;
  • O usuário possui um total de 3 chances. Se após 3 tentativas o usuário não acertar o número, o programa deverá informar que ele não conseguiu adivinhar e mostre pro usuário qual era o "número escondido".

OBS: O programa pode começar com um número pré definido, mas é interessante que posteriormente seja implementando usar um número randômico.

Vamos começar o desafio?

Escolhi um número...

Você consegue adivinhar que número é esse?
